# Battleship Project — Dead Reckoning

A polished browser-based reimagining of Battleship with three playable modes:

- **Classic** — original 10×10 rules and the traditional 5–4–3–3–2 fleet.
- **Rogue Fleet** — a 12×12 board with diagonal, hooked, trident, and zig-zag ships.
- **Commander** — tactical energy, sonar, torpedoes, and area barrages.

The game runs entirely in the browser with no build step or external JavaScript dependencies.

## Play locally

Open `index.html` in a modern browser.

## Deployment

Every push to `main` deploys the site through GitHub Pages.
